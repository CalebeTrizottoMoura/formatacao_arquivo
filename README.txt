INSTRUÇÕES DE USO

Este é um arquivo de instruções para o script Python fornecido. Por favor, siga as orientações abaixo para utilizar o script corretamente.


INSTALAÇÃO

Para executar o script, você precisa ter o Python instalado em seu sistema. Você pode baixar e instalar o Python a partir do site oficial: https://www.python.org/downloads/


DEPENDÊNCIAS

O script depende de algumas bibliotecas Python adicionais. Você pode instalar todas as dependências necessárias executando o seguinte comando:

pip install unidecode

Certifique-se de estar conectado à internet para que o pip possa baixar e instalar as bibliotecas corretamente.


EXECUÇÃO

Para executar o script, siga estas etapas:

1. Abra um terminal ou prompt de comando.
2. Navegue até o diretório onde o script Python está localizado.
3. Execute o código.


FORNECIMENTO DO CAMINHO DO ARQUIVO DE ENTRADA

Ao executar o script, você será solicitado a fornecer o caminho do arquivo que deseja processar. Certifique-se de fornecer o caminho correto do arquivo de entrada.


FORNECIMENTO DO CAMINHO DO NOVO ARQUIVO DE SAÍDA

Após o processamento do arquivo de entrada, você precisará especificar o caminho e o novo nome do arquivo de saída. Certifique-se de fornecer o caminho completo e o novo nome do arquivo de saída, incluindo o formato desejado.


CONTRIBUIÇÃO
